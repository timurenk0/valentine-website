Welcome to website launch guide!

After downloading .zip file of your website you need to complete following steps:

  1. Download and Organize Your Website Files:
    Ensure that all the necessary files for your website, including HTML, CSS, JavaScript, and any other assets, are organized in a dedicated folder on your local machine
  2. Prepare Your GitHub Account:
    If you haven't already, sign up for a GitHub account. GitHub is a platform for hosting your code repositories.
  3. Create a New Repository:
    Once logged in to GitHub, create a new repository by clicking on the "New" button. Give your repository a meaningful name that reflects your website.
  4. Upload Your Files to GitHub:
    After creating the repository, follow the instructions to upload your website files. Instead of zipping and uploading, select all the files in your website folder and upload them directly. This prevents potential issues that may arise from uploading an entire folder at once. Use drag & drop if not all the files were uploaded Here click on uploading an existing file link and upload your files
  5. Commit Your Changes:
    Once the files are uploaded, commit your changes to the repository. Provide a meaningful commit message that describes the changes you've made.
  6. Configure Repository Settings for GitHub Pages:
    Go to the "Settings" tab of your repository. Scroll down to the "GitHub Pages" section. In the "Branch" dropdown menu, select the branch that contains your website files (typically "main" or "master"). Then click "Save".
  7. Wait for GitHub Pages Deployment:
